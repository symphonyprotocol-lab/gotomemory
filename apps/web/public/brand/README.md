# gotomemory logo

This logo package uses the selected gotomemory mark: a privacy lock, connected memory nodes, and a forward arrow path. The shape communicates "your AI memory goes with you" while keeping the local-first privacy signal prominent.

## Files

- `gotomemory-logo-source.png` - source logo supplied by the project owner.
- `gotomemory-mark.svg` - SVG wrapper for the source logo.
- `gotomemory-mark-flat.svg` - alias wrapper for the source logo.
- `gotomemory-logo.svg` - horizontal mark plus wordmark.
- `gotomemory-ai-concept.png` - archived copy of the selected concept image.

Generated PNG exports use the same basename with size suffixes.

## Palette

- Deep ink: `#162033`
- Signal teal: `#00B8A9`
- Private coral: `#FF4D49`
- Recall gold: `#FDB52A`
- White: `#FFFFFF`

## Usage

Use the generated PNG icon files for app icons, favicons, extension icons, and square social avatars. Use `gotomemory-logo.svg` for website headers, README headers, and presentation title slides. Keep the mark clear of text or UI by at least one small node diameter.
